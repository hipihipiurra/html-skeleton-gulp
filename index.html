Skeleton

A html base structure for front-end development, automatized by Gulp.
With Skeleton you can:
  • Store sass variables on _variables.scss
  • Simplify all your content stylist with footer.scss, header.scss and content.scss
  • Call your scripts just pushing files into the folder /js -- it minifies itself!
  • Call your css just pushing files into the folder /css -- it minifies itself!
  • Use Font Awesome icons.
  • Use Bootstrap library for anything.
This version is a fork of HTML Skeleton Gulp.js project.
